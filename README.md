# Hanzo Fi Payment Dashboard

Web UI for managing payment routing, connectors, and analytics. Control center for the Hanzo Fi Payment Switch.

## Features

- Connector management (add, configure, enable/disable)
- Payment analytics and reporting
- Routing rule configuration
- Refund and dispute management
- API key management
- Multi-merchant support

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:9000
```

## Configuration

Set the Payment Switch API URL:

```bash
export APIBASEURL=http://localhost:8080
```

## Development

```bash
npm install
npm run dev      # Development server
npm run build    # Production build
npm test         # Run tests
```
